# Vetores
- Forma de trabalhar uma variável para guardar vários dados do mesmo tipo;

# Listas
- Permitem tipos de dados diferentes armazenados em uma variável;

## Tipos de listas
- Ligadas: Cada nó da lista sabe apenas quem é o próximo nó na ordem;
- Duplamente ligadas: Cada nó da lista sabe quem é o próximo nó na ordem e quem foi o nó anterior na ordem;

# Padrão Singleton
- Permite que uma classe tenha apenas uma instância durante a execução da aplicação;
