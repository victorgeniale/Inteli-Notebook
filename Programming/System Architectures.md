# Fatores na arquitetura
- Componentes: Módulos de responsabilidade específica no software (Representados por blocos)
- Propriedades: Particularidades de cada componente;
- Conectores: Forma como os componentes se comunicam entre si para a realização de tarefas;

# Tipos de arquitetura
> As arquiteturas podem ser agrupadas de acordo com o seu gênero e estilo

Gêneros: Comercial, comunicacional, autorial, financeiro, entretenimento...
Estilos.: Centrado em dados, fluxo de dados, chamada e retorno, orientada a objeto...
